# linux-shell-assignment
##overview 
This assignment helped me understand Linux better. I learned how to use commands to manage files and processes, and how to write scripts for tasks like backups and monitoring. I also practiced using GitHub to save and share code. 
##Script descriptions
> backup_directory.sh
Purpose: Creates a timestamped backup of a directory.

>monitor_resources.sh
Purpose: Periodically logs CPU and memory usage.

>auto_download.sh
Purpose: Automates downloading files via wget or curl.
